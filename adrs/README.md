# ADR Repository

Architecture Decision Records generated by the ADR Agent, organized by Cloud Service
Provider. Folders are created on demand as ADRs are filed.

```
adrs/
├── aws/
│   ├── compute/        storage/       database/{sql,nosql,warehouse}/
│   ├── networking/     serverless/    containers/
│   ├── security-identity/   analytics/   messaging/   monitoring/
├── gcp/   (same taxonomy)
└── azure/ (same taxonomy)
```

`index.json` (generated) is the flat list the UI reads. Each ADR is a Markdown file named
`ADR-XXXX-<slug>.md` with YAML front-matter.
